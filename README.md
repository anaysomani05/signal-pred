# Signal Pred

Live at [signal-pred.vercel.app](https://signal-pred.vercel.app)

Prediction market aggregator built with Next.js. Pulls live markets from Polymarket and Kalshi into a single feed with probability gauges, price changes, and category filters.

## Stack

Next.js 16 · React 19 · Tailwind CSS 4 · TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
